  1. Making Groups from the Continuous One thing that’s really cool about faceting is that you can use cut_*() functions on continuous variables to make facets by groups of continuous variables. To see what I mean, try cut_interval(1:10, n = 5) See how it makes five bins of even width? We use cut_interval() or other cut functions with faceting like so facet_wrap(vars(cut_interval(some_variable))).
ggplot(data = ice, mapping = aes( x =Extent)) +
  geom_histogram(bins = 30) +
  facet_wrap(vars(Month_Name,cut_interval(Extent, n = 2)))

With the original data, plot sea ice by year, with different lines (oh! What geom will you need for that?) for different months. Then, use facet_wrap and cut_interval(Month, n=4) to split the plot into seasons.

ggplot(data = ice, mapping = aes( y = Extent)) +

You have an issue here specifying your groups. The question asks you to plot sea ice by year with one line for each month so we can compare how ice extent has changed at the same points each year. To do this, you have to add the group aesthetic to your ggplot call like so:

year_plot <- ggplot(data = ice,
       mapping = aes(x = Year, y = Extent, group = Month)) +
  geom_line()
